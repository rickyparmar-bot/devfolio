# Contributing to DevFolio

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/devfolio.git`
3. **Create** a feature branch: `git checkout -b feature/amazing`
4. **Make** your changes
5. **Commit** with descriptive messages
6. **Push** to your fork
7. **Open** a Pull Request

## Guidelines

- Follow existing code style
- Add comments for complex logic
- Test your changes before submitting
- Update documentation if needed

## Ideas for Contributions

- 🌈 New themes
- 📊 Additional stat widgets
- 🐛 Bug fixes
- 📝 Better documentation
- 🎮 Interactive UI improvements
